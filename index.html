Sobre

O CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos HTML e XML. Com o passar dos anos, o CSS evoluiu significativamente, introduzindo novos valores e propriedades que permitem aos desenvolvedores web criar layouts mais flexíveis, responsivos e visualmente atraentes.

Entre os novos valores e propriedades do CSS, destacam-se o Flexbox e o Grid Layout, que facilitam a criação de layouts complexos sem a necessidade de hacks ou truques. O Flexbox permite alinhar e distribuir espaço entre itens em um contêiner, enquanto o Grid Layout oferece uma maneira poderosa de criar layouts bidimensionais.

Display

Define o elemento como um flex container, tornando os seus filhos flex-itens. O display:flex torna o elemento um flex container automaticamente transformando todos os seus filhos diretos em flex itens.

Flex-Direction

Define a direção dos flex itens. Por padrão ele é row (linha), por isso quando o display: flex; é adicionado, os elementos ficam em linha, um do lado do outro. A mudança de row para column geralmente acontece quando estamos definindo os estilos em media queries para o mobile. Assim você garante que o conteúdo seja apresentado em coluna única.

Flex-Wrap

Define se os flex itens devem ou não quebrar para a próxima linha quando não houver espaço suficiente. Por padrão, os itens não quebram e podem esticar o container. O uso de flex-wrap: wrap; permite que os itens se movam para a próxima linha, mantendo a responsividade.

Flex-Flow

Um atalho para flex-direction e flex-wrap, permitindo definir ambas as propriedades em uma única linha. Por exemplo, flex-flow: row wrap; define a direção como linha e permite que os itens quebrem para a próxima linha.

Justify-Content

Define como os itens são distribuídos ao longo do eixo principal. Os valores comuns incluem flex-start, flex-end, center, space-between e space-around. Isso permite um controle preciso sobre o alinhamento e o espaçamento dos itens dentro do container flex.

Align-items

Define como os itens são alinhados ao longo do eixo transversal. Os valores comuns incluem flex-start, flex-end, center, baseline e stretch. Isso permite um controle preciso sobre o alinhamento dos itens dentro do container flex.
